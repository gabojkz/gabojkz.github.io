How to publish a Hugo site in Github

5 min read London, UK

Intro

Why Hugo? I recommend it beacuse is the easiest way to generate a static website in my opinion. In this post I will show you how to start using hugo and how to deploy the site using github pages

Requirements

  1. Github Account

  2. New repository with the following name: [your username].github.io.

  3. Create another github repo but this time with a different name

  4. Install Hugo

  5. Pick a Hugo theme

Let’s start

With Hugo already installed we can start working on the site locally, to create new site you can use the following commands.

1

hugo new site [site name] && cd $1

Installing Hugo theme

Before we move forward we need to install a theme there are many you can choose from:

https://github.com/gohugoio/hugoThemes

The one I am using is this one:

https://themes.gohugo.io/hugo-casper-two/ (now I am using https://themes.gohugo.io/hugo-vitae/)

Ready to post

1

hugo

This command will create a new folder inside your project called /public this basicall means that hugo took all the content inside project and transform that into files that github can understand and render as a website.

cd public/
git init

# !IMPORTANT add as a origin the repository
# with the username username.github.io
# similar to:
# git@github.com:gabojkz/gabojkz.github.io.git
git remote add origin https://github.com/user/repo.git
git remote -v

# this will show the origin if is properly setup
git add --all
git commit -m "First commit"
git push

If you have done this properly you can now go to https://your_username.github.io/, and see the content there.

You can create a new repo in the root of you website where hugo is, This means that you will have two git repositories in the same project one to post the files hugo will generate and the another one to track the hugo project.

The content inside the folder public will go to one branch and the content inside the upper folder will go to another one.

That is my current setup I hope it give you an idea of how to setup Hugo and if you have any recommendations or comments please let me know.

Good luck.
